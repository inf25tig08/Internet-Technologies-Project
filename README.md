## Project Presentation

This project was developed for the Internet Technologies course and focuses on the evolution of multimedia. The website explores how digital media has changed over time, from traditional formats to modern technologies such as streaming platforms, virtual reality and artificial intelligence.

The project was developed using HTML5, CSS3, JavaScript, XML and XSD.

## User Interface
Site Map
- Home (index.html)
- Digital Media Transformation (media.html)
- Entertainment, Gaming and Virtual Reality (gaming.html)
- Artificial Intelligence and the Future of Multimedia (future.html)

## Interface Design

The website uses a simple navigation menu available on all pages. Content is organized into semantic sections and articles to improve readability and accessibility.

## Product Description

The website presents the historical evolution of multimedia technologies and their impact on society.

Main topics include:
- Evolution of video formats (VHS, DVD and Streaming)
- Evolution of music distribution (Radio to Spotify)
- Evolution of photography
- Gaming and Virtual Reality
- Artificial Intelligence in multimedia
- Website Availability
